Hostel Order System (Switch-Case)


📊 Logic Overview
This project demonstrates an interactive store interface using selection structures:
Numeric Selection: The user enters an integer (1, 2, or 3) to select an item from the menu.
Switch-Case Routing: Based on the choice variable, the program executes specific code blocks for Juice, Coffee, or Tea.
Variable Initialization: Key variables for pricing (e.g., juicePrice, coffeePrice) and taxation (salesTax) are predefined for future calculation expansion.
Input Validation: A default case is used to notify the user if an unrecognized numeric choice is entered.



🛠️ Features
Structured Menu: Provides a clear, centered visual menu for the user using tab escapes.
Modular Pricing: Separate integer variables allow for easy price updates in the source code.



🚀 Execution & Environment Guide
To run this project, follow the instructions below:

1. Recommended IDEs 
Dev-C++
Visual Studio Code
Online Compilers: Use OnlineGDB or Programiz to run directly in your browser.

2. Simple Execution Procedure
(Step 1): Setup the Project
create a file named Hostel_Order_System.cpp.
Copy the source code into your editor.

(Step 2): Compile the Code
In Dev-C++: Press F11 to Compile and Run.
In Terminal: Use g++ Hostel_Order_System.cpp -o OrderSystem then ./OrderSystem.

Step 3: Test the Output

Select an item by typing 1, 2, or 3 to see the confirmation message.
